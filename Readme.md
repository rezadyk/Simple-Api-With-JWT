# Simple API with Json Web Token Authentication #

This is BackEnd service for Json Web Token Authentication (JWT).

## Including Library ##
- express
- mongodb
- jsonwebtoken
- morgan
- dotenv (to store secrete key)

### To run this service : ###
1. Clone this Repository
2. Run npm install
3. Change Key in *.env* variable
4. npm run dev
